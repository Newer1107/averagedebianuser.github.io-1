SkibidiOS 0.01 download link (mega.nz, 1.1GB download, 3GB decompressed)

The file is a raw QEMU .img, which can probably be flashed onto a USB stick. It's 3 gigabytes, even though the install only takes up about 2 gigabytes. This is so that you can actually install things when trying out in QEMU.

Note that this is STILL UNFINISHED. I would release this later but decided to push out an early release.

"This sucks." Yes, it does, it's a joke.

skibidwm keybinds: (will make this more readable later)

  • Win+Enter = open terminal emulator
  • Win+(Number) = switch to Nth workspace
  • Win+Q = close focused window
  • Win+G = hide top bar
  • Win+Shift+Q = end xorg session (go back to tty)
  • Win+J = focus left window
  • Win+K = focus right window
  • Win+H = resize focused window to the left
  • Win+L = resize focused window to the right
  • Win+F = zoom
  • Win+Space = spawn dmenu
